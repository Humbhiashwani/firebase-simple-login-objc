Inherits from NSObject

Overview

The FAUser class is a wrapper around the user metadata returned from the Firebase auth server. It includes a (userId, provider) combo that is unique, as well as the token used to authenticate with Firebase.

It may include other metadata about the user, depending on the provider used to do the authentication.
